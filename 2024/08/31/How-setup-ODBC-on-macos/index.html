How setup ODBC on macos

Created2024-08-31|Updated2024-08-31
|Post Views:

Install ODBC management software

Because macos itself does not have ODBC management software, you need to install ODBC management software before installing the specific ODBC driver

I have installed an ODBC management software called iODBC. The official website of this software is https://www.iodbc.org/dataspace/doc/iodbc/wiki/iodbcWiki/Downloads#Mac%20OS%20X

Be sure to install the version that is compatible with your operating system.

Install ODBC

Because I need to connect to the MySQL database on the cloud server, I need to go to the MySQL official website to find the corresponding version of the ODBC driver

the official website link is https://downloads.mysql.com/archives/c-odbc/

Configuration

After the installation is complete, you need to perform the corresponding configuration. This configuration operation is performed through the ODBC management software we installed earlier. There is a lot of relevant information on the Internet about the configuration details, so I will not go into details here.

Solve the error

After the configuration is complete, when you try to connect to the database in Excel, it usually shows that the corresponding driver file cannot be loaded. This is because the Excel program does not have permission to access the installed driver file. So we need to make some changes to the corresponding driver file.

copy the entire folder

change the content of ODBC.ini file in /Library/ODBC/

Driver = /Library/ODBC/mysql-connector-odbc-8.0.30-macos12-arm64/lib/myodbc8w.so

after those small adjustment, the database link will be successfully.
